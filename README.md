## Setup Guide:
  * You should have these installed:
      * python 2.7.x - Use official python installer
      * pip  `sudo easy_install pip`
      * tox  `pip install tox`

## Start server:
    * run ./setup.sh
    # run ./runserver.sh start or ./runserver.sh start debug for debug mode

## Stop server:
    * run ./runserver.sh stop

## UI:
  * cd react/
  * npm run start
